# ClassroomFucker

班上的人天天拿一体机看一些特别糖的片，各种什么奶龙、锻刀大赛、华强卖瓜啥的，我被严重精神污染。
因此我写了一个这个项目，干扰一体机工作，这样再也没有人能够放那些唐人片了。

全项目使用AI完成，没人喜欢干查win32api这种黑奴工作。

## 有关信息

程序生效的时间是这样的：

- 课间
    - 08:45 - 08:53
    - 09:40 - 10:08
    - 10:55 - 11:03
    - 14:25 - 14:33
    - 15:20 - 15:33
    - 16:20 - 16:28
- 长休息
    - 11:50 - 13:38 午休
    - 17:15 - 18:28 晚休

## 需求

本程序需要兼容Windows7, 10, 11，第一次运行使用管理员权限运行

本程序使用u盘进行初次运行，进程名使用`Win32Service`伪装
运行后程序立即将程序本体拷贝到`C:/ProgramData/ForYes/Win32Service.exe`下，添加开机自启（管理员权限）。
第一次从u盘安装后5分钟内不做任何事情，不会干扰一体机（便于我不被学生发现，不然我运行了一个程序之后电脑就发疯了，肯定会被发现），
之后开始执行这些策略：

- 程序常驻后台，开机自启，不创建任何窗口。
- 每1秒检测一次本地时间，在课间时间内不断重复地kill, start explorer.exe，在长休息时间内20秒后关机。
- 如果课间、长休息时间内检测到u盘进入，立即格式化u盘（恶心学生的pe盘重装策略）

使用C++编写此程序。
